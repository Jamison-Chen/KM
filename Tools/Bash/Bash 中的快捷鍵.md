### `Control` + `C`

送出 SIGINT (signal interrupt) 訊號，以打斷目前終端機的 Process。

### 方向鍵 $\uparrow$

顯示上一次輸入的指令。

### `Tab`

輸入指令時，可以透過 `Tab` 自動補齊未打完的 sub-directory name 或 file name。

### `Command` + `K` 或 `Ctrl` + `L`

在 Unix 系統的 terminal 使用此快捷鍵可以清空目前的 terminal。
