### Expression

Expression 指的是一段「可以產生一個 value」的「values、operators 與 functions 的組合」，比如：

```plaintext
# Python
1 + 1

abs(x) + floor(int(y)) + 1

// JavaScript
a === b ? 0 : 1  // ternary expression
```

### Statement

Statement 指的是一段「執行某項任務」的程式碼，比如：

```plaintext
# "if statement" in Python
if a > 0:
	...

# "for loop statement" in Python
for i in range(10):
	...

# "assign statement" in Python
a = 10
```
