# Parsing URL

因為我們是以瀏覽器訪問網站，所以瀏覽器會對我們輸入在網址列中的文字做一些前處理，前處理包括：

- 辨認輸入的內容是不是一個網址
    - 如果是的話就會直接造訪該網址；否則就會進入搜尋模式（將文字丟給搜尋引擎）
- 將每個 non-ASCII 的字元轉換為「16 進制表示的 UTF-8 byte sequences」並在每個 sequence 前加上一個 `%`
    - e.g. `Hello你好` → `Hello%E4%BD%A0%E5%A5%BD`
