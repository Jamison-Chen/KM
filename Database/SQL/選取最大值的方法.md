範例資料：

**car**

| brand | made_year | price |
| ---- | ---- | ---- |
| Tesla | 2019 | 100000 |
| Ford | 2000 | 50000 |
| Ford | 1980 | 35000 |
| Toyota | 2010 | 70000 |
| Tesla | 2021 | 100000 |

### 法一：在 subquery 中使用 `MAX()` 作為篩選條件

```PostgreSQL
SELECT * FROM cars
WHERE price = (SELECT MAX(price) FROM cars);
```

Output:

```plaintext
  brand  | made_year | price
---------+-----------+-------
 Tesla   |      2019 | 100000
 Tesla   |      2019 | 100000
(2 rows)
```

### 法二：降冪排序後，選擇第一列

```PostgreSQL
SELECT * FROM cars
ORDER BY price DESC
LIMIT 1;
```

Output:

```plaintext
  brand  | made_year | price
---------+-----------+-------
 Tesla   |      2019 | 100000
(1 rows)
```

你會發現，當最大值有不只一個 row 時，法一會全數列出，法二則只會列出第一筆，因此兩種方法應視需求使用。
