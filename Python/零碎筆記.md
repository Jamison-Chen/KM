### The Walrus Operator (`:=`)

在 Python 3.8 之後，可以在 `if`、`for` 與 `while` statement 中使用 Walrus Operator，舉例來說：

```Python
# Before Python 3.8
l = len(s)
if l > 2:
    print(l)

# Since Python 3.8
if (l := len(s)) > 2:
    print(l)
```
